# Processador 8 bits em Verilog

Projeto acadêmico desenvolvido na disciplina de Arquitetura e Organização de Computadores (CEFET-MG).

## Implementação de um processador de 8 bits com suporte a instruções básicas como:
- Load
- Store
- Desvios condicionais
- Desvios incondicionais
- Halt

## Tecnologias
- Verilog
- ModelSim
- Quartus
- FPGA Cyclone II

## Estrutura do projeto
- src/: módulos principais (ALU, registradores, unidade de controle, datapath)
- testbench/: arquivos de teste

## Status
Projeto acadêmico finalizado para fins educacionais.
